TestGen is a test-generating and editing application. It allows Instructors to quickly create, organize, transfer, and customize questions and answers from Wiley’s testbank content. Tests can be prepared for use with a wide variety of learning management systems and for distribution in various digital and printed formats. TestGen is available for Windows and Mac and is utilized worldwide across all subjects.

It is the only application that edits equations without a separate editor and with its random number generator you can create test questions that have different text or number values each time you use them. Its powerful search and sort functions let you quickly locate questions and arrange them in the order you prefer.

Testbanks & Exam Building

  • Easy test creation using a variety of methods, including drag-and-drop, copy/paste, multiple selections, and filtering. Ability to copy/paste fields from other applications.
  • Full word processing capabilities including a built-in spell check plus additional key exam building functionality.
  • Create tests that can be printed or displayed in a web browser for online testing.
  • Global font, styles, and size control.
  • Test questions can be presented in various forms such as multiple-choice, short-answer, true/false, matching, essay & fill-in, vocabulary, and bimodal. Up to 25 multiple-choice answers per question.
  • Prepare tests for use with learning management systems (LMS) such as Blackboard, Canvas, D2L, Moodle and Sakai.
  • Testbank organization is based on the associated textbook.
  • Global changes to the test such as add/remove answer blanks, multiple choice layout and answer style changes.
  • Test items can be presented in two-column format.
  • Tests appear on screen exactly as they do on paper. No need for extra preview option.
  • Custom Headers, Footers and Notes may be added to the test and can include page number, date, time, as well as custom text.
  • Testbank data may include virtually unlimited accessibility compliant Alt Text characters including for function plots and transparent graphics.
  • Customizable information on answer keys (explanations, fields, answer text, etc.).
  • Automatic renumbering of items and adjustments to answer key when you insert a new item or delete items on the test.
  • The order of the questions and answers can be scrambled each time a student takes a test.
  • Up to 25 scrambled versions of a test can be printed at once.

Question Editing

  • Questions may be selected from a testbank included with each TestGen package for rapid test creation, or tests may be created from an original set of questions.
  • Selection of the actual questions may be manual or randomly generated. Random question selection can be keyed to fields and/or ranges of questions.
  • Unlimited questions can be added into a test one at a time or in groups and question size is unlimited with smart split of questions across page.
  • On-screen editing of test questions and customizable test layouts and printing options to fit an Instructor’s course and style of teaching and testing.
  • The testbank questions can be either static or algorithmic and can be made available for use on a desktop or via the web.
  • No limitations on where questions can be moved within the testbank and they may be recorded using drag-and-drop or cut-and-paste directly in the testbank editing window.
  • Numerous descriptors help classify and filter questions including page reference, difficulty, topic, skill, objective, comments, explanation, etc.

Math & Algorithmic Capabilities

  • The TestGen application was the first to excel in mathematics and the presentation of true algorithmic data.
  • Questions can include stylized text, symbols, graphics, and equations that are inserted directly into questions using built-in mathematical templates.
  • The application includes an algorithmic number and text generator capability that, at the Instructor's discretion, produces different text or number values creating numerous iterations of the same test question for administration control.
  • Built-in table editor, equation editor, math grapher, and symbol palettes.
  • The expression editor allows the direct insertion of variables, symbols, templates, math notations and expressions. Special characters can also be added via symbol palettes and copied from other applications.
  • Tab support provided to simplify the alignment of columnar data.
  • Built in function plotter supports up to 15 functions per graph and full control over backgrounds, range and domain, labels, size of lines and scaling.

Graphics, Exports & more

  • Support for a variety of graphics formats including JPG, PNG, BMP, TIF, and GIF.
  • Instructors may create Cartesian, polar, and number line graphs using a built-in Graphing tool.
  • Graphics can be black and white, gray scale, or full color.
  • Support for both Hi and Low-resolution graphics for printing and screen.
  • Graphics can be inserted as a character or transparently.
  • Tests may be exported in numerous formats including RTF (for use with MS Word), PDF, "HTML with Images", "HTML with MathML", and various LMS and proprietary formats.
  • Autogenerated math Alt Text is included in the RTF, LMS and "HTML with Images" exports.
  • The "HTML with MathML" export option generates MathML for native TestGen math expressions and templates.
